# MachineLearning-and-Big-Data-Processing
## About
Practical notebooks from the Machine Learning and Big Data Processing Course

## Topics Covered
- End-to-end ML workflow: data cleaning, EDA, train/val/test splits,
  cross-validation, feature selection, hyperparameter tuning
- Classical ML: regression, decision trees, random forest, XGBoost
- Scikit-learn Pipelines for reproducible workflows
- Real sensor data preprocessing and time series forecasting
- Experiment tracking and hyperparameter sweeps with Weights & Biases
- Deep learning: feedforward networks, CNNs (CIFAR-10),
  autoencoders and denoising autoencoders (MNIST)
- NLP and Transformers: word embeddings, self-attention from scratch,
  transformer encoder for text classification, decoder-only transformer
  for text generation

## Tools & Libraries
Python, scikit-learn, PyTorch, torchvision, XGBoost, pandas, NumPy,
Weights & Biases
